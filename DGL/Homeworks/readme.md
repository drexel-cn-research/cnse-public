# DGL Homework Assignments

This folder contains the seven Deliberate Gap Learning (DGL) homework assignments used in the study reported in *Beyond Lecture and Lab: Deliberate Gap Learning as an AI-Era Homework Framework* (IEEE Transactions on Education, submitted 2026). Each assignment targets a topic intentionally left uncovered in lecture, requiring students to bridge the gap through a structured AI inquiry dialogue.

| HW | Topic | File |
|----|-------|------|
| 1 | CISC/RISC complexity & power | [HW1.md](HW1.md) |
| 2 | Cache hierarchy & memory locality | [HW2.md](HW2.md) |
| 3 | Interrupts & IoT power | [HW3.md](HW3.md) |
| 4 | RISC-V ISA extensions | [HW4.md](HW4.md) |
| 5 | Carry propagation & subtraction | [HW5.md](HW5.md) |
| 6 | Specialized execution units | [HW6.md](HW6.md) |
| 7 | CPU exceptions & privilege modes | [HW7.md](HW7.md) |

HW 1–4 cover processor architecture (ISA) topics; HW 5–7 cover hardware design topics. All assignments were deployed in an undergraduate systems architecture course (N = 105) and share a common rubric that assesses investigative process quality rather than answer correctness.
